# TiXL Curriculum Guide: Beginner Level

## Overview
This curriculum guide provides a comprehensive framework for teaching TiXL to beginners. The program is designed for students with basic computer literacy who want to learn data analysis and visualization fundamentals.

## Program Structure

### Duration: 8 weeks (32 contact hours)
### Prerequisites: Basic computer literacy, familiarity with spreadsheet software
### Class Size: 20-25 students maximum
### Delivery Mode: In-person, online, or hybrid

## Learning Objectives

By the end of this program, students will be able to:

### Week 1-2: Foundation Skills
- Navigate the TiXL interface confidently
- Import and export data files in various formats
- Understand basic data types and structures
- Create simple visualizations (bar charts, line graphs, pie charts)

### Week 3-4: Data Management
- Clean and prepare datasets for analysis
- Handle missing and inconsistent data
- Perform basic calculations and create formulas
- Organize data using filters and sorting

### Week 5-6: Visualization Mastery
- Select appropriate chart types for different data
- Customize visualizations with colors, labels, and formatting
- Create combination charts and multi-axis visualizations
- Design basic interactive dashboards

### Week 7-8: Applied Analysis
- Interpret visualizations to extract insights
- Create presentations with data-driven recommendations
- Understand statistical concepts in context
- Develop a portfolio of data analysis projects

## Weekly Curriculum Breakdown

### Week 1: Introduction to TiXL and Data Visualization

**Learning Objectives:**
- Understand the TiXL interface and navigation
- Import and explore sample datasets
- Create first visualizations

**Session 1 (2 hours): TiXL Interface Overview**
- Welcome and course overview
- TiXL interface tour and navigation
- Hands-on practice with sample data
- Assignment: Explore TiXL with provided dataset

**Session 2 (2 hours): Data Import and Basic Charts**
- Importing data from CSV, Excel, and databases
- Understanding data structures and types
- Creating basic charts (bar, line, pie)
- Assignment: Create three basic charts from provided data

### Week 2: Chart Fundamentals and Customization

**Learning Objectives:**
- Master chart creation and basic customization
- Understand when to use different chart types
- Apply visual design principles

**Session 1 (2 hours): Chart Types and Selection**
- Overview of chart types and their purposes
- When to use bar charts vs. line charts vs. pie charts
- Hands-on practice with multiple chart types
- Assignment: Analyze dataset and select appropriate chart types

**Session 2 (2 hours): Customization and Design**
- Formatting charts with colors, fonts, and styles
- Adding titles, labels, and annotations
- Design principles for effective visualization
- Assignment: Customize three charts using design principles

### Week 3: Data Preparation and Cleaning

**Learning Objectives:**
- Identify and handle data quality issues
- Clean and transform data for analysis
- Create calculated fields and derived metrics

**Session 1 (2 hours): Data Quality Assessment**
- Identifying missing, duplicate, and inconsistent data
- Using data validation and error checking
- Data profiling and summary statistics
- Assignment: Audit provided dataset for quality issues

**Session 2 (2 hours): Data Cleaning Techniques**
- Removing duplicates and handling missing values
- Data type conversion and formatting
- Creating calculated fields and formulas
- Assignment: Clean dataset and add calculated fields

### Week 4: Advanced Data Operations

**Learning Objectives:**
- Filter and sort data effectively
- Create pivot tables and summaries
- Perform basic statistical analysis

**Session 1 (2 hours): Filtering and Sorting**
- Advanced filtering techniques
- Multi-level sorting and grouping
- Creating data subsets for analysis
- Assignment: Create filtered views of dataset

**Session 2 (2 hours): Pivot Tables and Summaries**
- Creating and customizing pivot tables
- Calculating totals, averages, and percentages
- Grouping and aggregating data
- Assignment: Create comprehensive data summary

### Week 5: Interactive Visualizations

**Learning Objectives:**
- Create interactive charts and dashboards
- Add filters and controls to visualizations
- Design user-friendly dashboard layouts

**Session 1 (2 hours): Interactive Charts**
- Adding interactivity to charts
- Creating drill-down capabilities
- Interactive filtering and selection
- Assignment: Add interactivity to existing charts

**Session 2 (2 hours): Dashboard Creation**
- Dashboard layout and design principles
- Combining multiple visualizations
- Creating cohesive dashboard themes
- Assignment: Build first interactive dashboard

### Week 6: Advanced Visualization Techniques

**Learning Objectives:**
- Create combination charts and multi-axis visualizations
- Use advanced chart types effectively
- Apply advanced formatting and styling

**Session 1 (2 hours): Combination Charts**
- Creating charts with multiple data series
- Dual-axis charts and their applications
- Overlaying different chart types
- Assignment: Create combination charts from complex datasets

**Session 2 (2 hours): Advanced Chart Types**
- Scatter plots and correlation analysis
- Heat maps and geographical visualizations
- Waterfall charts and flow diagrams
- Assignment: Explore advanced chart types

### Week 7: Data Analysis and Interpretation

**Learning Objectives:**
- Interpret visualizations to extract insights
- Identify trends, patterns, and outliers
- Apply statistical concepts practically

**Session 1 (2 hours): Trend Analysis**
- Identifying trends in time series data
- Calculating moving averages and growth rates
- Forecasting simple trends
- Assignment: Analyze trends in provided dataset

**Session 2 (2 hours): Comparative Analysis**
- Comparing data across categories and time periods
- Identifying patterns and relationships
- Statistical concepts in practice
- Assignment: Conduct comparative analysis

### Week 8: Presentation and Portfolio Development

**Learning Objectives:**
- Create compelling data presentations
- Tell stories with data
- Build a professional portfolio

**Session 1 (2 hours): Data Storytelling**
- Principles of effective data storytelling
- Structuring presentations with data
- Creating narrative flow with visualizations
- Assignment: Develop presentation outline

**Session 2 (2 hours): Final Project and Portfolio**
- Students present final projects
- Peer review and feedback
- Portfolio review and improvement
- Course evaluation and next steps

## Assessment Methods

### Formative Assessment (40% of grade)
- Weekly assignments and practice exercises (20%)
- Class participation and engagement (10%)
- Peer review activities (10%)

### Summative Assessment (60% of grade)
- Mid-term project: Data analysis of provided dataset (20%)
- Final project: Comprehensive analysis with presentation (30%)
- Final exam: Practical skills assessment (10%)

## Instructor Preparation

### Pre-Course Requirements
- TiXL Certified Instructor certification
- Review of all course materials and datasets
- Setup of classroom/lab environment
- Preparation of backup plans for technical issues

### Teaching Tips
- Use real-world examples relevant to students' backgrounds
- Encourage questions and peer collaboration
- Provide immediate feedback on student work
- Connect new concepts to previously learned material

### Common Challenges
- Varying levels of computer literacy among students
- Different learning paces and styles
- Technical difficulties with software
- Building confidence in data analysis

### Solutions
- Provide additional support for struggling students
- Offer extension activities for advanced learners
- Have backup datasets and offline exercises ready
- Create a supportive learning environment

## Technology Requirements

### Hardware
- Computers with minimum 8GB RAM, modern processor
- Large monitor or dual monitor setup recommended
- Reliable internet connection for online resources

### Software
- TiXL latest version with educational license
- Office suite for document creation
- Web browser for online resources

### Accessibility
- ADA-compliant classroom setup
- Screen reader compatibility for visually impaired students
- Captioned videos and transcripts for hearing impaired students
