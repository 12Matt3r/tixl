## ğŸ¯ Quick Start

### 1. Choose Your Path

```bash
# For beginners - start with the Getting Started tutorial
git clone https://github.com/tixl3d/examples-getting-started.git

# For intermediate developers - browse by category
# Check out the Graphics examples:
git clone https://github.com/tixl3d/examples-graphics.git

# For advanced users - download the complete collection
git clone https://github.com/tixl3d/examples-complete.git
```

### 2. Run an Example

```bash
# Navigate to an example
cd examples/graphics/pbr-materials

# Restore dependencies
dotnet restore

# Build and run
dotnet run --project src/PbrMaterials.csproj

# Or use the provided build script
./build-and-run.sh
```

### 3. Learn and Modify

- Read the example documentation
- Experiment with parameters
- Try combining examples
- Build something new!

